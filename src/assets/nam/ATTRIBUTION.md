(https://github.com/sdatkinson/NeuralAmpModelerPlugin, MIT, © Steven Atkinson)
in Pulp-native view/canvas code. The UI *code* is an independent Pulp
implementation — none of NeuralAmpModelerPlugin's iPlug2/`IGraphics` code is
used. The image and font assets below are reused under their own licenses so the
recreation is faithful.
